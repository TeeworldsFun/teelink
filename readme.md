H-Client its a mod for the game Teeworlds created and maintained by unsigned char*.

Please visit http://hclient.wordpress.com for up-to-date information about 
the mod.

- libnotify (Only Linux)
- gdk_pixbuf-2.0 (Only Linux)
- libopus
